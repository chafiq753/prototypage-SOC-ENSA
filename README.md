# prototypage-SOC-ENSA
Exemplaire d'un Dashboard SOC  pour la visualisation et la surveillance  de la platefrome universitaire de l'ENSA
## Pages
- Dashboard – Vue d'ensemble ( des tableaux et des courbes  pour visualiser ) 
- Visualisation des Logs ( Journalisation )
- Centre d'Alertes (  Indication des alertes avec leur états ( en cours , résolus , ouvert)
- Export CSV (  Exportation des actions du jours dans un fichiers CSV )  

## Technologies
HTML, CSS, Chart.js
